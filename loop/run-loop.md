# ループの回し方

ループ本体は [`run-loop` スキル](../.claude/skills/run-loop/SKILL.md) に入っている。**毎回プロンプトを貼る必要はない** —— スキルを呼ぶだけ。

## 手動で一周
```
/run-loop
```

## 定期実行（人間が最後にタイマー化）
- ラップトップを開いている前提のローカル繰り返し:
  ```
  /loop 30m /run-loop
  ```
- まず triage だけ低頻度で様子を見たいなら:
  ```
  /loop 30m /triage
  ```
  progress.md の更新を目視確認し、慣れたら `/run-loop` に広げる。
- ラップトップを閉じても走らせたい常駐は `/schedule` でクラウド実行のルーティンを作る。

**まず短い間隔・低頻度で開始し、出力を見ながら調整する。** いきなりフル自動 + PR作成に乗せない。マージは常に人間。
